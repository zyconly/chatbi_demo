## 🎯 设计系统

### 颜色系统
| 颜色变量 | 值 | 用途 |
|---------|-----|------|
| `primary-color` | #2563EB | 主操作、按钮、链接 |
| `primary-dark` | #1E40AF | 深蓝（渐变用） |
| `secondary-color` | #7C3AED | 创建按钮、强调 |
| `bg-white` | #FFFFFF | 卡片背景 |
| `bg-light` | #F8FAFC | 轻浅背景 |
| `bg-gray` | #F1F5F9 | 输入框背景 |
| `text-primary` | #1F2937 | 主文本 |
| `text-secondary` | #6B7280 | 次要文本 |
| `text-tertiary` | #9CA3AF | 辅助文本 |
| `border-color` | #E5E7EB | 边框线 |
| `success-color` | #10B981 | 成功状态 |

### 间距系统
```
xs: 4px    sm: 8px    md: 16px   lg: 24px   xl: 32px
```

### 排版规范
- **大标题** (24px, 700) - 页面标题
- **标题** (20px, 600) - 导航标题
- **副标题** (16px, 600) - 卡片标题
- **正文** (14px, 400) - 内容文本
- **标签** (12px, 400) - 辅助信息
- **小文字** (11px, 400) - Meta 信息

### 圆角规范
- 容器：8-12px
- 按钮/输入框：6-8px
- 头像：6-8px

## 🚀 如何使用

### 打开设计文件
```bash
# 在 Pencil 设计工具中
1. 打开 chatbi-complete-pages.pen
2. 查看文档左侧的页面列表
3. 点击切换不同页面预览
```

### 与开发对接
1. **导出设计令牌** - 复制所有变量值到 CSS 变量
2. **导出组件** - 使用 Pencil 导出设计规范文档
3. **UI 组件参考** - 前端开发参照设计实现组件
4. **设计稿标注** - 所有间距、字体大小都已标准化

### 设计令牌导出示例
```css
:root {
  --primary-color: #2563EB;
  --secondary-color: #7C3AED;
  --bg-white: #FFFFFF;
  --text-primary: #1F2937;
  --spacing-md: 16px;
  --spacing-lg: 24px;
  /* ... 更多变量 */
}
```
